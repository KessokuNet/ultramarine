# KessokuNet Ultramarine Shade

This is an Ultramarine Shade for KessokuNet users, with stuff that people might find useful.

## Why not upstream to Ultramarine?

Ultramarine Shades are designed to be custom user derivations of the base Ultramarine bootc system, and such, Shades are intended
to be the upstream derivation image for custom user layers. KessokuNet Shade is one such layer, intended to be a stable daily-driver
runtime for KessokuNet users, which is out of scope for the base Ultramarine bootc image.
