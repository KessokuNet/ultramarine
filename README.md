# KessokuNet Ultramarine Image

This is an Ultramarine Image for KessokuNet users, with stuff that people might find useful.

## Why not upstream to Ultramarine?

While I (korewaChino) am the founder of the Ultramarine project, the contents of this custom image is deemed out of scope
for the base image, and is not intended to be used as the base image for Ultramarine.
It is intended to be an example custom image for Ultramarine users wanting to build and maintain their own custom images.
This is the intended way to customize Ultramarine bootc; by maintaining your own image based on the upstream base,
and then using downstream as your actual OTA source.
